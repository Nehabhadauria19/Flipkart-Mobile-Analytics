# 📊 Flipkart Mobile Analytics

## 📌 Project Overview

This project analyzes Flipkart mobile dataset to understand pricing trends, brand performance, and customer preferences.

---

## 🎯 Objective

* Analyze mobile pricing and ratings
* Identify top-performing brands
* Find value-for-money products

---

## 🛠 Tools Used

* SQL (PostgreSQL)
* Python (Pandas, Matplotlib)
* Power BI
* Excel

---

## 📂 Project Structure

```id="k0n12s"
data/ → dataset  
sql/ → SQL queries  
python/ → Python analysis  
dashboard/ → Power BI dashboard  
```

---

## 📊 Analysis Performed

* Brand-wise average rating analysis
* Price segment (Budget, Mid, Premium) analysis
* Top reviewed products identification
* Price vs rating relationship

---

## 📈 Key Insights

* Mid-range segment has highest number of products
* Premium phones do not always have higher ratings
* Some budget phones provide strong value for money
* Brand performance is consistent across segments

---

## 📊 Dashboard

Power BI dashboard includes:

* KPI metrics (Total Products, Avg Rating)
* Brand performance chart
* Price segment distribution
* Price vs rating analysis
